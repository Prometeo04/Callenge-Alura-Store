# Callenge-Alura-Store
Análisis de Datos de una franquicia de tiendas

Estructura del proyecto
1. Carga de datos

Cada tienda tiene un archivo con su información de ventas y fletes.
El código los carga utilizando pandas.

2. Cálculo de métricas

Para cada tienda se calcula:

Total vendido por categoría

Costo de envío promedio (flete_promedio)

Otras métricas que tú agregues después

3. Visualización

Se genera un gráfico de barras donde se comparan los fletes promedios de las cuatro tiendas.

Este gráfico permite identificar:

Qué tienda tiene los costos de envío más altos

Diferencias logísticas entre sucursales

Oportunidades para optimización de costos
